Overview

Popup notifications provide a convenient way to receive messages from the application.

When enabled, notifications will appear in the lower-right portion of your screen in response to events that occur in the application. Examples of events that might trigger a popup include:

  • A file download has completed
  • The network connection is lost
  • An error was encountered when reading a layer file

Popup notifications may contain hyperlinks that will perform specific actions related to the notification. If you wish to perform the associated action, simply click on the appropriate hyperlink.

The behaviour of popup notifications can be configured through the preferences window.
